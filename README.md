## Live - <a href="https://naflanmohamed.github.io/Interstellar-Link/" >Interstellar Link - VISIT Omega Vespera </a>

Interstellar Link is an single-page concept website for a futuristic space travel agency. It features a high-performance HTML5 Canvas starfield, interactive particle effects, and a CSS-only 3D planet simulation.

## Features

### Visuals

- Starfield engine with more than 1,000 stars and depth.
- Hyperdrive mode with camera shake.
- Mouse driven nebula particles.
- CSS planet and rings built with gradients and animations.

### Look and Feel

- Glassmorphism UI.
- Neon theme with Cyan (#00f3ff) and Purple (#bc13fe).
- Fully responsive with Tailwind CSS.

### Functionality

- Booking modals for travel tiers.
- Remembers selected package.
- Toast notifications on form submit.

## Tech Stack

- HTML5
- CSS3
- JavaScript ES6
- Tailwind CSS CDN
- Canvas API 2D
- Orbitron and Rajdhani fonts
